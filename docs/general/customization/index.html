Customization

Mijin offers minimal customizations to focus on usability. Yet, the default theme setting can be overridden to give a specific aspect to the UI..

Tailwind CSS

Mijin comes with it's Tailwind CSS presets. These presets can be override from your tailwind.config.js file:

// tailwind.config.js

const mijin = require('mijin/tailwind-preset.js');
const { colors, borderRadius } = require('tailwindcss/defaultTheme');

module.exports = {
  presets: [
    mijin,
  ],
  theme: {
    extend: {
      // override a component border-radius
      borderRadius: {
        button: borderRadius.sm,
        form: borderRadius.sm,
      },
      // override a color
      colors: {
        primary: colors.pink,
      },
    },
  },
};

    Plugin

    The plugin can be initialized with options. For example, to change the default prefix from Mj to M, load the plugin with the following option

    import Vue from 'vue';

Vue.use(Mijin, {
  prefix: 'M',
})
